字符串前缀u,r,b,f

u，后面字符串以Unicode格式编码，常用于防止中文出现的乱码问题。

r，纯文本？去掉反斜杠机制，常用于正则表达式re

b，bytes数据类型，常用于网络编程种，浏览器、服务器的数据传输

f，表示字符串内支持大括号内的python表达式
